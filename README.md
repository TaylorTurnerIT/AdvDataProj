# Impact of 9/11 on U.S. Transportation Analysis

## Overview

This project performs a time series analysis to evaluate the immediate and long-term impacts of the September 11, 2001 attacks on major modes of U.S. transportation, including air, rail, and vehicle travel.

## Key Files

* **`Sept11Analysis.ipynb`**: Jupyter Notebook containing the Python code for data processing, statistical summary, and visualization.
* **`Sept11Travel.csv`**: The primary dataset containing monthly travel metrics (Air Revenue Passenger Miles, Rail Passenger Miles, and Car Vehicle Miles Traveled) from 1990 to 2004.

## Usage

Run the `Sept11Analysis.ipynb` notebook to reproduce the time series plots and summary statistics. The analysis focuses on detecting trend shifts and structural breaks in travel behavior following the events of 2001.
